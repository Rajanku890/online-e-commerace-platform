# online-e-commerace-platform
# 🛒 Online E-Commerce Platform

A fully functional Online E-Commerce Web Application that allows users to browse products, add items to cart, and securely place orders.  
This project demonstrates full-stack web development skills including frontend design, backend integration, and database management.

---

## 📌 Project Overview

The Online E-Commerce Platform is designed to simulate a real-world shopping website with user authentication, product management, cart system, and admin control panel.

It is developed as an academic project for B.Tech Computer Science Engineering.

---

## 🚀 Features

### 👤 User Module
- User Registration & Login
- Browse Products
- Search & Filter Products
- Add to Cart
- Remove from Cart
- Place Orders
- Order History

### 🛠 Admin Module
- Admin Login
- Add / Update / Delete Products
- Manage Users
- Manage Orders
- Dashboard Overview
